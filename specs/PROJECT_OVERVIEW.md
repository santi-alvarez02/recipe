# Recipe 3D App

An immersive 3D recipe application inspired by [jesse-zhou.com](https://jesse-zhou.com).

## Vision
Create a visually stunning, interactive recipe management app that feels like stepping into a modern kitchen. Users should feel delighted by every interaction.

## Core Features
1. **3D Recipe Storage** — Browse recipes in an immersive 3D environment
2. **AI Recipe Generation** — Create new recipes with OpenAI
3. **URL Import** — Extract recipes from any website
4. **Voice Input** — Hands-free ingredient entry

## Design Principles
- 60fps on mid-range devices
- Mobile-first responsive 3D
- Smooth, physics-based animations
- Accessible fallback for non-WebGL browsers

## Tech Stack
- Next.js 15 + TypeScript + React 19
- React Three Fiber + drei
- Tailwind CSS + Framer Motion
- Supabase + OpenAI API

## Inspiration
[jesse-zhou.com](https://jesse-zhou.com) — Award-winning cyberpunk ramen shop portfolio featuring immersive 3D environments, neon-lit aesthetics, and smooth interactions built with Three.js.
